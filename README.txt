Projet: Portfolio statique
Contenu: HTML, CSS, JS, images SVG (placeholders) et un CV en pdf.
But: modèle simple et responsive de portfolio prêt à personnaliser.

Fichiers:
- index.html
- css/style.css
- js/main.js
-fonts/fonts/RobotoCondensed-Italic-VariableFont_wght.ttf
- assets/profile.svg
- assets/project-1.svg
- assets/project-2.svg
- assets/project-3.svg
- assets/Alex_Camel_CV.pdf


Comment utiliser:
1. Décompresse le fichier ZIP.
2. Ouvre `index.html` dans ton navigateur.
3. Modifie le contenu, remplace les SVG par tes images, et adapte les textes.
4. Pour rendre le formulaire opérationnel, connecte-le à un backend (Flask, Netlify functions, Formspree, etc.)

Besoin d'une version Flask ou React ? Dis-le-moi et je te fournis la version complète.
