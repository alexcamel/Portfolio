Projet: Portfolio statique
Contenu: HTML, CSS, JS, Flask, images SVG (placeholders) et un CV en pdf.
But: modèle simple et responsive de portfolio prêt à personnaliser.

Fichiers:
- templates/
    - index.html
- static/
    - css/style.css
    - js/main.js
    - fonts/RobotoCondensed-Italic-VariableFont_wght.ttf
    - assets/profile.png
    - assets/project-1.svg
    - assets/project-2.svg
    - assets/project-3.svg
    - assets/Alex_Camel_CV.pdf
- app.py


Comment utiliser:
1. Décompresse le fichier ZIP.
2. Ouvre `index.html` dans ton navigateur.
3. Modifie le contenu, remplace les SVG par tes images, et adapte les textes.
4. Pour rendre le formulaire opérationnel, connecte-le à un backend (Flask)
